How it works ?
  • User must provide access to location service. Based on location, User's group is decided
  • User registers with Google ID. User is required to be at home during the registration process. During registration, user's group is identified. If you are not in your home group then it will be difficult for you.
  • After registration, user can see the group ID on home page of the app
  • User will see slots available for booking. Active slots can be clicked to book. If slot is grayed (disabled) then it is full. i.e. number of people allowed to go out already booked this slot. You must choose a different slot.
  • User will also see booked slots. If you click on the slot, you can delete the booked slot.
  • You must go outside only during your booked slot. You must show the booked slot to police at shop to be able to purchase.
